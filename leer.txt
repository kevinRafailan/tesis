- Configurar su conexion local con:
password = '' (es decir sin password)
usuario = 'root'
base de datos = 'campoescuela'

los datos deben ser los mismos para todos asi evitamos que la conexion
cambie cada vez que alguientra trabajo en el proyecto.

- En la carpeta "vista" crear sub carpetas para cada vista
ejemplo
vistas->inventario
vistas->cuentas
vistas->usuarios
etc...
para manterner el orden

utilizar carpeta "base" dentro de vista... para crear nuevas vistas.

usar un prefijo en cada uno de los archivo de modelo y controlador
ejemplo:
ym_actualizar_producto.php
ym_crear_usuario.php

al igual que el caso anterior, crear carpetas para cada cosa que este relacionada o sean
de la misma "categoria"
